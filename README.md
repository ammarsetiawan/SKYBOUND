# SKYBOUND 🎮

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Platform: Web / PC](https://img.shields.io/badge/Platform-Web%20%2F%20PC-blue.svg)]()

**SKYBOUND** adalah sebuah game platformer 2D yang menantang di mana pemain harus melompat dari satu platform ke platform lainnya, menghindari rintangan berbahaya, dan mendaki setinggi mungkin untuk menyelesaikan level. Didukung dengan visual pixel art yang menarik dan gameplay yang adil namun menantang, game ini akan menguji ketangkasan dan waktu reaksi kamu!

## ✨ Fitur Utama

*   **Gameplay Platformer Klasik:** Kontrol yang responsif untuk melompat dan bermanuver di udara.
*   **Rintangan Menantang:** Berbagai macam jebakan seperti duri (*spikes*) dan musuh unik di setiap sudut.
*   **Desain Level Dinamis:** Desain platform vertikal yang memaksa pemain untuk terus bergerak ke atas.
*   **Estetika Pixel Art:** Visual retro yang memanjakan mata lengkap dengan animasi karakter yang halus.
*   **Audio & Musik Imersif:** Efek suara dan latar musik yang meningkatkan atmosfer permainan.

---

## 🎮 Cara Bermain

Tujuan utama game ini adalah mencapai garis akhir (*finish line*) di setiap level tanpa terkena rintangan.

### Kontrol Karakter
| Aksi | Tombol Kontrol |
| :--- | :--- |
| **Bergerak Kiri** | Panah Kiri (`←`) |
| **Bergerak Kanan** | Panah Kanan (`→`) |
| **Melompat / Lompat Ganda** | `Spacebar` atau Panah Atas (`↑`) |

> ⚠️ **Tips:** Hati-hati dengan penempatan duri di platform sempit. Perhitungkan waktu lompatanmu dengan matang!
